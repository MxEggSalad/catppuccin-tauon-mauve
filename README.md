![preview](Catppuccin_Mauve_Tauon.png)

## Installation
Download [Catppuccin-Mauve.ttheme](Catppuccin-Mauve.ttheme)

Open `Menu > Settings > Function > Open data folder`.

Place `Catppuccin-Mauve.ttheme` inside `theme` folder.
